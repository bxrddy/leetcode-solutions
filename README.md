## Getting Started

### Prerequisites

Ensure you have Python installed. If not, download and install Python from [python.org](https://www.python.org/downloads/).

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/bxrddy/leetcode-solutions.git
   ```

2. Navigate to the repository
    ```
    cd leetcode-solutions
    ```

3. Running a solution

    To run a specific solution, navigate to its directory and run the Python script.  

    ```
    python3 solutions/easy/problem_number/problem_name.py
    ```
